# react-template
This template contains the following keywords:

- Typescript
    - TSLint
- React
    - Redux
        - Typescript FSA
        - Redux Thunk
    - React Router
    - Styled Components
- Webpack
    - Awesome TypeScript loader
    - Webpack Html Plugin

## How to use
### Setup
- Install [Node](https://nodejs.org/en/download/current/), [Yarn](https://yarnpkg.com/lang/en/docs/install)

### Build
    $ yarn install
    $ yarn build

### Start development web server

    $ yarn dev
